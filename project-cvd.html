Multimodal CVD Risk Prediction

Siamese DenseNet201 for Cardiovascular Disease Risk Assessment

← Back to Projects

Main Technology Stack

Python, PyTorch, Siamese DenseNet201, OpenCV, Scikit-learn, SHAP / Grad-CAM (XAI), Kaggle T4 GPU.

Brief Description

A multimodal deep learning pipeline that predicts cardiovascular disease (CVD) risk by jointly analyzing retinal fundus images and carotid intima-media thickness (CIMT) data using a Siamese DenseNet201 architecture, trained on the China-Fundus-CIMT dataset. The model achieved an AUC of 0.9168, surpassing the benchmark reported in the original source paper. The project also includes ablation studies, statistical significance testing, and publication-ready methodology and figures written in IEEE JBHI format.

Challenges Faced

  • Aligning and fusing two very different data modalities (retinal images and CIMT vitals) into a single learnable architecture.
  • Training a deep Siamese network efficiently within Kaggle's free T4 GPU memory and time limits.
  • Designing ablation experiments and statistical tests rigorous enough for IEEE JBHI-level review.

Future Improvements

  • Validate the model on external, multi-center datasets to test generalizability.
  • Explore lighter backbone architectures for faster inference in clinical settings.
  • Extend explainability analysis with counterfactual methods alongside Grad-CAM/SHAP.
